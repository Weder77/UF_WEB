## Setup
To run this project, you need to install mysql with brew.

Brew installation :

```
$ // Ignore this step if you have already brew
$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" 
```

Mysql installation :

```
$ brew install mysql
$ mysql.server start
```
